## 🧠 Overview

This project implements an **AI-based Environment Monitoring and Prediction System** that:

- Retrieves **real-time weather + air quality data**
- Performs automatic preprocessing + feature extraction
- Trains AI/ML models to predict environmental trends
- Generates alerts for unsafe conditions
- Provides an interactive **Streamlit Dashboard UI**

---

## ✨ Key Features

### 🌦 Real-Time Weather Monitoring
Fetches live data such as:

- Temperature  
- Humidity  
- Pressure  
- Wind Speed  

---

### 🌫 Air Quality Monitoring
Tracks pollutants and AQI levels:

- PM2.5 / PM10  
- CO, NO₂, SO₂  
- Overall AQI Index  

---

### 🤖 AI Prediction Engine
Uses Machine Learning models to:

- Forecast temperature trends  
- Detect anomalies  
- Predict environmental risks  

---

### 🚨 Smart Alert System
Triggers warnings when thresholds exceed safe limits.

---

### 📊 Interactive Streamlit Dashboard
A complete web interface for:

- Live monitoring  
- Graph visualization  
- Predictions  
- Risk alerts  

---
